HTML Tag References

Tables
Tag Name Description
<table> Table Wrapper for all HTML tables
<tr> Table Row adds row to table
<td> Table Data Adds data points to table
<th> Table Heading Adds headings to table
<colspan> Column Span Used for data that spans multiple columns
<rowspan> Row Span Used for data that spans multiple Rows
<tbody> Table Body Section of the table containing all data
<thead> Table Head Used to section the header of the table
<tfoot> Table Footer Used to section off the bottom part of the table
